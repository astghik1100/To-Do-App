### Run Locally

To run **Word Scramble** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/astghik1100/To-Do-App.git
```

Windows:

```bash
git clone https://github.com/astghik1100/To-Do-App.git
```
